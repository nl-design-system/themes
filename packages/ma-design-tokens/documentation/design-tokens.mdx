## Common design tokens (basis)

<DesignTokensTable
  tokens={tokens.filter((token) => token.path[0] === "basis")}

## Component Design Tokens

<DesignTokensTable
  tokens={tokens.filter(

  tokensDefinition={tokensDefinition}
></DesignTokensTable>

## Figma only Design Tokens

<DesignTokensTable
