# NL Design System design tokens Open Formulieren

The design tokens for Open Forms are maintained in the
[`open-formulieren/design-tokens`](https://github.com/open-formulieren/design-tokens)
repository and installed via the
[`@open-formulieren/design-tokens`](http://npmjs.com/package/@open-formulieren/design-tokens)
npm package.
